# Intro to Cloud Services

This is just a file to get a basic overview and to look at one or two examples
of services.

## Examples of Cloud Services

- Azure
- Amazon Web Services (AWS)
- Google Cloud

### Examples of sub-sections of Azure

- Azure App Service
  - Complete package of services for launching an application in the cloud
  - Includes automation of distribution and scalability
  - Also includes security-functionality like zero trust-principles (via Entra
    ID?)
  - Ease of use in focus for all levels from study/hobby to full enterprise
- Azure MySQL
  - Open-source compatibility in focus
  - Let's you plan for maintenance easily with scheduled maintenance windows
  - Automatic backups and point-in-time restore available if crashes happen
  - Automatic patching and maintenance of underlying hardware and structure
- Azure Entra ID
  - collects the identities and access control for an application
  - Is built into pretty much every microsoft product like Azure and Office365
